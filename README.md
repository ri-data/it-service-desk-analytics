# IT Service Desk Analytics

## Overview
Analysis of 100,000 IT service desk tickets to identify performance patterns, 
resolution trends, and customer satisfaction drivers.

## Tools
Python | MySQL | Power BI

## Business Questions Answered
1. Average resolution time by Priority
2. Agent Group performance comparison
3. Contact Source efficiency
4. Monthly ticket volume trends
5. Customer dissatisfaction by Topic
6. Support Level distribution
7. Agent interactions vs satisfaction (bonus impact)

## Key Findings
- Critical tickets resolved in 0.83 hours vs 3.83 hours for Low priority
- IT Support has lowest satisfaction rate despite average resolution speed
- Hardware Failure tickets generate highest dissatisfaction at 33.73%
- August volume drop reflects incomplete month data
